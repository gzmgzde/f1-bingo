Welcome to the **F1 Bingo**, created for the **UWSU Motorsports Society**! This virtual bingo game is perfect for watch parties, allowing society members to mark race events in real-time and compete for 3 in a row.

## How to Play

1. **Open the game** in a browser (mobile or desktop).
2. **Enter your username** in the input box at the top.
3. **Tap squares** on the board as events happen during the F1 Qatar GP race.
4. **Get 3 in a row** (horizontally, vertically, or diagonally) to win.
5. When you win, your username will be **logged** (and can be sent to a server if configured).
6. Use the **Reset Marks** button if you want to unmark squares without changing the card.

## Features

* **Fixed 3x3 board:** Everyone plays with the same board for fairness.
* **Markable squares:** Click or tap to mark events as they occur.
* **Winner detection:** Alerts when a player completes 3 in a row.
* **Track winners:** Currently logs winners in the console; can be connected to a backend.
* **Responsive design:** Works on desktop and mobile.
