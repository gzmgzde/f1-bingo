Welcome to the **F1 Bingo**, created for the **UWSU Motorsports Society**! This virtual bingo game is perfect for watch parties, allowing society members to mark race events in real-time and compete for marking all 9 squares.

## How to Play

1. **Open the game** in a browser (mobile or desktop).
2. **Enter your username** in the input box at the top.
3. **Tap squares** on the board as events happen during the F1 Qatar GP race.
4. **Get 3 in a row** (horizontally, vertically, or diagonally) to complete a line.
5. To win, mark all 9 squares. 
6. When you win, your username will be **logged**.
7. Use the **Reset Marks** button if you want to unmark squares without changing the card.

## Features

* **Fixed 3x3 board:** Everyone plays with the same sized board with randomised cards for fairness.
* **Markable squares:** Click or tap to mark events as they occur.
* **Winner detection:** Alerts when a player completes 3 in a row.
* **Track winners:** Currently logs winners in the console.
* **Responsive design:** Works on desktop and mobile.

## Hosting

The game is hosted on **GitHub Pages**, with **Firebase** database.

## Notes

* No login is required; usernames are self-entered.
* Refreshing the page will reload different board.
* Designed specifically for **UWSU Motorsports Society events**, but can be adapted for other watch parties or motorsports events.
